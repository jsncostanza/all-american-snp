Accessibility Statement

We're committed to making our site usable for everyone. If you run into a barrier, please let us know — we want to fix it.

our commitment

Equal Access for All Customers

All American Sandwiches N Pizza is committed to ensuring that our website is accessible to people with disabilities. We believe everyone should be able to browse our menu, learn about our specials, and reach us with the same ease, whether they're using a screen reader, keyboard navigation, voice control, or any other assistive technology.

We work to follow the recommendations of the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA, published by the World Wide Web Consortium (W3C). These guidelines explain how to make web content more accessible for people with disabilities and more user‑friendly for everyone.

what we've done

Accessibility Features

This site has been built with the following in mind:

  • Descriptive alternative text on meaningful images so screen readers can convey them.
  • Semantic HTML structure (headings, lists, landmarks) so assistive technology can navigate the page.
  • Sufficient color contrast between text and background.
  • Responsive layout that scales cleanly on phones, tablets, and desktops, and supports browser zoom up to 200%.
  • Keyboard-accessible navigation and interactive elements.
  • Clearly labeled form fields and buttons.
we're still working

Known Limitations & Ongoing Work

Accessibility is an ongoing effort, not a one-time project. We're continually testing and improving our site, and we recognize that some areas may not yet fully meet the standards we aim for. If you encounter content that's difficult to access, please contact us using the details below — we'll work to address it promptly.

We also rely on a small number of third-party embeds (for example, the Google Maps location embed on our Contact page). We do not directly control the accessibility of third-party content, but we'll do our best to provide an alternative if you let us know it's a problem for you.
